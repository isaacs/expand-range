## Example usage

```js
var expand = require('{%= name %}');

expand('1..3')
//=> ['1', '2', '3']

expand('5..8')
//=> ['5', '6', '7', '8']

expand('00..05')
//=> ['00', '01', '02', '03', '04', '05']

expand('01..03')
//=> ['01', '02', '03']

expand('000..005')
//=> ['000', '001', '002', '003', '004', '005']

expand('a..e')
//=> ['a', 'b', 'c', 'd', 'e']

expand('A..E')
//=> ['A', 'B', 'C', 'D', 'E']
```

Pass a function as the last argument to customize the expansions:

```js
var range = expand('a..e', function (str, i) {
  return String.fromCharCode(str) + i;
});

console.log(range);
//=> ['a0', 'b1', 'c2', 'd3', 'e4']
```
